# @nexex/orderbook

## Description

Backend service of NexEx which maintain orderbook and sync between other peers within the same cluster and peers across ipfs pubsub.

For singleton usage, please refer to @nexex/cli 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
